# ectract files to text

## Purpose
Tools to extract text content from files for processing and indexing.

## Files
- `extract_texts.py` (main extractor)
- `filestotext.md` (usage guide)


## What these files do
extract_texts.py reads files and extracts text content for indexing; filestotext.md explains options.

This section explains purpose-level behavior and how they support your project.

## Importance
This section is important for building and running project components reliably.
## How to run
1. Install Python 3.8+.
2. Open terminal in `ectract files to text`.
3. Run `python extract_texts.py`.
4. Reference `filestotext.md` for detailed options.
