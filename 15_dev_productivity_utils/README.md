# 15_dev_productivity_utils

## Purpose
Developer productivity tools for code stats, doc generation, formatting, and project bootstrap.

## Files
- `code_stats.py` – count lines of code (multi-language)
- `doc_generator.py` – generate markdown API docs from docstrings
- `formatter.py` – format code (black + isort)
- `project_creator.py` – scaffold Python projects
- `dependency_checker.py` – check installed packages, detect outdated/missing
- `virtualenv_manager.py` – create/manage venvs
- `git_commit_analyzer.py` – analyze commit history
- `code_complexity_checker.py` – measure cyclomatic complexity (AST-based)
- `log_file_analyzer.py` – parse logs, detect errors/severity
- `config_loader.py` – load YAML/TOML/JSON/.env configs
- `environment_variable_manager.py` – manage env variables`


## What these files do
code_stats.py calculates code metrics, doc_generator.py builds docs, formatter.py formats code, project_creator.py scaffolds projects.

This section explains purpose-level behavior and how they support your project.

## Importance
This section is important for building and running project components reliably.
## How to run
1. Install Python 3.8+.
2. Open terminal in `15_dev_productivity_utils`.
3. Run any script, e.g. `python code_stats.py`.
