# 10_testing_utils

## Purpose
Testing and load test scripts for API and unit testing.

## Files
- `api_test_runner.py`
- `load_test.py`
- `unit_test_template.py`


## What these files do
api_test_runner.py runs API tests, load_test.py exercises performance tests, unit_test_template.py shows pytest structure.

This section explains purpose-level behavior and how they support your project.

## Importance
This section is important for building and running project components reliably.
## How to run
1. Install Python 3.8+ and dependencies (`pytest`, `requests` etc.).
2. Open terminal in `10_testing_utils`.
3. Execute `python api_test_runner.py` or `python load_test.py`.
4. Use `unit_test_template.py` as a template for your tests.
