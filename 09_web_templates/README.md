# 09_web_templates

## Purpose
Starter templates for web applications built with FastAPI, Flask, and React.

## Subfolders
- `fastapi_starter`
- `flask_starter`
- `react_dashboard`


## What these files do
Container for FastAPI, Flask, and React starters with web app scaffolding.

This section explains purpose-level behavior and how they support your project.

## Importance
This subtree provides web app starter templates that help bootstrap frontend and backend projects.
## How to run
1. Enter one of the subfolders.
2. Follow the README in that subfolder:
   - FastAPI: `pip install -r requirements.txt`, then `uvicorn main:app --reload`.
   - Flask: `pip install -r requirements.txt`, then `python app.py`.
   - React: `npm install`, then `npm start`.
