# 02_data_tools

## Purpose
Data ingestion, transformation, and utilities for common data workflows.

## Files
- `data_tools.py` (main data utilities)
- `quickinstallation.md` (setup guide)
- `transform_cfg.json` (example transform configuration)


## What these files do
Data import/export, transformation options, and JSON config-driven transformations for ETL workflows; transform_cfg.json defines mapping rules.

This section explains purpose-level behavior and how they support your project.

## Importance
This section is important for building and running project components reliably.
## How to run
1. Install Python 3.8+.
2. Open terminal in `02_data_tools`.
3. Run `python data_tools.py`.
4. Use `transform_cfg.json` for data pipeline configuration.
